CSS3

CSS é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML. Ele é usado para controlar a aparência visual de uma página da web, incluindo layout, cores, fontes e outros aspectos de design. O CSS3 é a terceira versão do CSS e inclui novos recursos e melhorias em relação às versões anteriores.
